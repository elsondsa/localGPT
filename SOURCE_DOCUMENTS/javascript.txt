Write a Javascript function "main()" that follows characteristics:

1. This function we write for certain plugins(http,sms). This function returns a next module Id to be executed based on some conditions.
2. There are special type of variables available in this function:
	1. Variables start from "$" symbol. These variables come from the above plugins and have values inside them. These are called global Variables.
	2. Response variables which also start with "$", but their names will be different based on their plugin type:
		a. If plugin is http, then the response variable is $response.
		b. If plugin is sms, then the response variable is $response.
	3. If you want to pass any variable to next Module id, then you can set the variable with setVariable function.
		For eg:
			let customerName = $response.data.customerName;
			setVariable('name' , customerName);
	4. You can also set a response Code. This response Code will help to understand what has happened in that particular code.
	   For eg:
			if($response.successCode == 200){
				let message = $response.successMessage;
				setVariable('message', message); //pass variables to next module
				return "a3456b" //next module Id
			}
			else if($response.successCode == 699){
				let message = $response.failureMessage;
				setVariable('message', message);
				return "b344ca" //another module Id
			}
			else{
				let message = "Sorry, your request cannot be processed";
				setVariable('message', message);
				return "a22344b"; //another module id
			}
3. Below is a sample code for a given Use-case.

Input: Write a code for the http plugin. The global variables are the customerType and customerRegion. If customer region is "india" and customerType is "premium" , then set the variable "balance" to $response.premiumBalance and return to a sms module ID. If customer region is "America", and customer Type is "premium" then set the variable "balance" to $response.premiumBalanceCheck and return to a different sms module ID. If customer is "india" and customerType is "generic", then set the balance to $response.indiaBalance and return to a email module. If customer is "ameria" and customerType is "generic", then set the balance to $response.americaBalance and return to the same email module. If none of the cases satisfy then go to a appEnd module with response code as E9000.

Code:

function main() {
  if ($customerType == 'generic') {
    if ($customerRegion == 'india') {
      setVariable('balance', $response.indiaBalance);
    }
    else {
      setVariable('balance', $response.americaBalance);
    }
    return "abbcc5"; //return to same email module
  }
  else if ($customerType == 'premium') {
    setVariable('balance', $response.premiumBalance);
    if ($customerRegion == 'india') {
      return "5d4ccf"; //a sms module
    }
    else {
      return "67ddef"; //different sms module
    }
  }
  else {
    setResponseCode('E9000');
    return "8990gdg"; //appEnd module Id
  }
}
